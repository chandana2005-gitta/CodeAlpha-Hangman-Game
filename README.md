# Hangman Game

## Overview

Hangman Game is a simple text-based Python game where the player guesses letters to uncover a hidden word. The player has a limited number of attempts to guess the word correctly before the game ends.

## Features

* Random word selection
* Letter-by-letter guessing
* 6 attempts for each game
* Win and lose conditions
* Beginner-friendly Python implementation

## Technologies Used

* Python
* Random Module

## How to Run

1. Install Python on your system.
2. Download or clone this repository.
3. Open a terminal in the project folder.
4. Run the command:

```bash
python hangman.py
```

## Sample Gameplay

```text
🎮 Welcome to Hangman!

Word: _ _ _ _ _ _
Attempts Left: 6

Guess a letter: a
```
